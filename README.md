# Rurudo My Beloved (chikenscrach.github.io)

這是 [csg](https://github.com/chikenscrach) 的個人首頁網站。

## 🚀 專案特點

- **響應式設計**：支援行動裝置與桌面瀏覽器。
- **PWA (Progressive Web App)**：支援離線檢視與安裝至主畫面。
- **動態視覺效果**：
  - 櫻花飄落背景 (Canvas 實作)。
  - 打字機效果 (Typewriter effect)。
  - 互動式卡片懸停效果。
- **彩蛋**：隱藏的 Konami Code 指令。

## 🛠️ 技術堆棧

- **HTML5** / **CSS3** (包含 CSS 變數、Flexbox)
- **Vanilla JavaScript** (ES6+)
- **Service Worker** (用於快取與 PWA 支援)

## 📂 檔案結構

- `index.html`: 網站主要結構。
- `styles.css`: 樣式表。
- `script.js`: 主要邏輯，包含櫻花特效、打字機效果與彩蛋。
- `sw.js`: Service Worker 腳本。
- `manifest.json`: PWA 設定檔。
- `image/`: 存放頭像與背景圖。
- `icons/`: 存放 PWA 圖示。

## 🖱️ 如何在地運行

1. 複製此儲存庫：
   ```bash
   git clone https://github.com/chikenscrach/chikenscrach.github.io.git
   ```
2. 直接開啟 `index.html` 或使用 Live Server (VS Code 擴充功能) 預覽。
